# Advanced English Quiz

An improved version of the basic English quiz.  
Now includes 5 questions, skip/stop options, and detailed performance statistics.

---

##  Features
-  5 diverse questions (grammar, vocabulary, etc.)  
-  Option to skip or stop early  
-  Statistics: attempts, accuracy, total score  
-  Better error handling and validation  

---

##  Tech Stack
- **Language:** Python 3  
- **Concepts:** loops, conditionals, functions, data structures  

---

##  How It Works
1. The user answers questions or skips them.  
2. Results are tracked in real time.  
3. The program outputs a detailed report at the end.  

Example:

Skipped 1 question.
Correct answers: 4/5
Accuracy: 80%

- Tracking and calculating user performance
